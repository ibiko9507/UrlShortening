Temel Uri kısaltma uygulamasıdır.
Fonksiyonalite olarak;
 - random Uri kısaltma
 - Kısaltılmış uri' ın uzun halini db'den getirme.
 - Manuel kısa uri girişi özelliklerine sahiptir.
